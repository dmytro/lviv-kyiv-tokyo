<br>
Here you can find the current status of the translations of Linux Mandrake
tools. You can retrieve the files from this page if you want to help
with the translations, just click on the percent number or the "-".
Then you can send the updated pages back to 
<a href="mailto:pablo@mandrakesoft.com">pablo@mandrakesoft.com</a>.
If you want to start translations for a language not yet listed here,
please <a href="mailto:pablo@mandrakesoft.com">write me</a> before
as they may be some few things to do before (and I have to add another
column to this page and ask for a nice flag t obe made :) )
<br>
It really isn't difficult at all to do some translations, however
there are some points you must be aware of
if you never worked with pofiles:<br>
<ul>
<li>never touch the <b>msgid</b> strings, only fill the <b>msgstr</b> ones
<li>when an entry is tagged as <b>fuzzy</b> that means it is probably
	incorrect; it has to be checked, corrected, and the <b>fuzzy</b>
	tag removed.
<li>you can use the command <b>msgfmt -v -o /dev/null file.po</b> to check
	for errors
</ul>
